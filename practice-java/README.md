# 🚀 kode-exp — Java Learning Repository

A structured, hands-on Java practice repo covering Core Java from the ground up.  
Every topic lives in its own package with focused, runnable programs.

---

## 📁 Repository Structure

```
kode-exp/
├── src/
│   └── coreJava/
│       ├── fundamentals/     → Printing, variables, escape sequences, CLI input
│       ├── operators/        → Arithmetic, relational, logical, bitwise, ternary
│       ├── loops/            → for, while, do-while, nested loops, break/continue
│       ├── string/           → String pool, literals vs new, common methods
│       └── oops/             → Classes, objects, constructors, inheritance, polymorphism
└── README.md
```

## ▶️ How to Run Any Program

```bash

```

---

## 📌 Conventions

- Each `.java` file = one focused concept
- Package matches the folder path: `package coreJava.<topic>;`
- Comments explain the *why*, not just the *what*
- File names use **PascalCase**
