# 📘 Core Java — Overview

Everything under `src/coreJava/` is organized into topic-based packages.  
Follow the roadmap below in order for the smoothest learning curve.

---

## 📚 Topics

| # | Package | What You'll Learn |
|---|---------|-------------------|
| 1 | [`fundamentals`](fundamentals/README.md) | Printing, variables, data types, escape sequences, CLI args |
| 2 | [`operators`](operators/README.md) | All Java operator types with real examples |
| 3 | [`loops`](loops/README.md) | for, while, do-while, nested loops, break & continue |
| 4 | [`string`](string/README.md) | String pool, common methods, immutability |
| 5 | [`oops`](oops/README.md) | OOP pillars — classes, objects, inheritance, polymorphism |

---

## 💡 Tips

- Run each file independently — every class has its own `main()`.
- Read the inline comments; they explain concepts step by step.
- Modify the programs and experiment — that's the best way to learn!
